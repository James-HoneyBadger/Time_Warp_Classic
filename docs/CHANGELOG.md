# Time_Warp - Changelog

## Current Version (Educational Interpreter)

Time_Warp is a simple educational programming environment supporting 9 programming languages with integrated turtle graphics. This project focuses on providing an accessible way to learn programming concepts across different paradigms.

- **Multi-Language Support**: PILOT, BASIC, Logo, Python, JavaScript, Perl, Pascal, Prolog, Forth
