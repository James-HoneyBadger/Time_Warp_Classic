## ğŸ¯ Quick Start

1. **Install dependencies**: `pip install -r requirements.txt`
2. **Run Time_Warp**: `python Time_Warp.py` (launches CLI by default)
3. **Try a sample program**: Load any `.pilot`, `.bas`, or `.logo` file from the `examples/` directory
4. **Use the CLI**: `python Time_Warp.py --cli examples/pilot/hello.pilot`
